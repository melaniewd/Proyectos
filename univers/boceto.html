CONSULTA



Un recurso frontend es cualquier tecnología, herramienta o archivo utilizado en el desarrollo de la interfaz de usuario de un sitio web o aplicación web. Su función principal es crear una experiencia de usuario atractiva, interactiva y funcional en el navegador web.

Algunas Principales tecnologias o herramientas utilizadas en el desarrollo de Frontend son HTML, CSS Y JavaScript. HTML se utiliza para estructural el contenido de una pagina web, CSS se utiliza para estilizar y diseñar la apariencia de la pagina y JavaScript se utiliza para agregar interactividad y funcionalidad dinamica Estas tecnologías son importantes porque permiten a los desarrolladores crear interfaces de usuario atractivas y funcionales que mejoran la experiencia del usuario y hacen que el sitio web o la aplicación sean más fáciles de usar y navegar.

La responsividad en el diseño web se refiere a la capacidad de un sitio web para adaptarse y mostrarse correctamente en una variedad de dispositivos y tamaños de pantalla. Dos técnicas para lograr la responsividad son:

  • Media queries: Permiten aplicar estilos diferentes según las características del dispositivo, como el ancho de la pantalla.
  • Diseño fluido: Utilizar unidades relativas como porcentajes en lugar de unidades fijas como píxeles para que los elementos se ajusten dinámicamente al tamaño de la pantalla.
  • HTML (Lenguaje de Marcado de Hipertexto): Se utiliza para estructurar y organizar el contenido de una página web.
  • CSS (Hojas de Estilo en Cascada): Se utiliza para estilizar y diseñar la apariencia visual de una página web.
  • JavaScript: Se utiliza para agregar interactividad y funcionalidad dinámica a una página web.

    • La accesibilidad en el diseño frontend es crucial para garantizar que todas las personas, incluidas aquellas con discapacidades, puedan acceder y utilizar un sitio web o aplicación. Dos prácticas para mejorar la accesibilidad son:

  • Uso de etiquetas semánticas HTML: Ayuda a los lectores de pantalla a comprender la estructura y el contenido de la página.
  • Contraste adecuado: Asegurarse de que haya suficiente contraste entre el texto y el fondo para facilitar la lectura para personas con discapacidades visuales.

    • Los frameworks frontend son conjuntos de herramientas y bibliotecas predefinidas que proporcionan una estructura y funcionalidades comunes para el desarrollo frontend. Dos ejemplos son:

  • React.js: Un framework de JavaScript desarrollado por Facebook para la creación de interfaces de usuario interactivas.
  • Angular: Un framework de JavaScript desarrollado por Google para la creación de aplicaciones web de una sola página (SPA).

    • La principal diferencia radica en que los CMS están orientados al contenido y ofrecen funcionalidades específicas para la gestión de este, mientras que los frameworks frontend se centran en proporcionar herramientas para el desarrollo de la interfaz de usuario y la lógica de la aplicación.

  • CMS (Sistema de Gestión de Contenidos): Proporciona una interfaz de administración para crear, editar y gestionar contenido web. Ejemplos: WordPress, Drupal.
  • Framework: Proporciona una estructura y funcionalidades comunes para el desarrollo frontend, pero no incluye una interfaz de administración. Ejemplos: React.js, Angular
